# JSON Row Header Hint

This small VS Code extension shows the header string (from the first array item) as a faint inline hint when clicking inside subsequent array rows in JSON files where the first top-level array element is an array of strings.

Build & Run

```bash
cd vscode-extension-json-row-hint
npm install
npm run compile
# Open this folder in VS Code and press F5 to run the extension in Extension Development Host
```
